Репозиторий - это папка с сайтом
Комит - это название заливаемого обновления
Пуш - отправить на репозиторий гит-хаба

1.Назначить для папки удалённый репозиторий кликнув правой кнопкой по нужной папки и открыть в ней GIT BASH. 
Там вводим:
$ git clone "ПУТЬ______.git"
Сам путь берём на сайте, при создании нового репозитория.
2. Создаём нужные файлы в этой папке. 
3. В GIT_BASH, чтобы добавить файл вводим:
$ git add "самфайл"
либо добавить все файлы
$ git add . 
4. Закомитеть
$ git commit -m "названиекомита"
5. Пушем. Добавляем всё на удалённый репозиторий
$ git push origin master
